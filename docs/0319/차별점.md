# 프로젝트 차별점

## 핵심 컨셉

> 전적 사이트는 "내가 얼마나 잘하는지 보는 곳", 이 프로젝트는 **"내가 얼마나 잘하는지 남한테 보여주는 곳"**

- 전적 사이트가 아님. tracker.gg, op.gg 같은 분석 도구와는 완전히 다른 카테고리.
- 전적을 기반으로 FC FUT 카드 디자인을 생성하는 **"자랑 도구"**.
- 디자인이 메인. 결국 공유되는 건 이미지 한 장이고, 그 한 장이 예뻐야 퍼진다.

## 핵심 기능

### 1. FC 카드 디자인

- 발로란트 전적 데이터를 기반으로 FC FUT 스타일 카드 생성
- 티어별 카드 디자인 차등 (레디언트, 다이아 등 높은 티어일수록 화려한 디자인)
- 스탯에 따른 알고리즘 점수 산출

### 2. AI 요원 이미지 생성 (NanoBanana2)

- 공식 일러스트를 그대로 쓰면 모든 유저의 카드가 동일 → 개성 없음
- NanoBanana2를 활용해 공식 일러스트 화풍을 유지하면서 다양한 포즈 생성
- 같은 요원이라도 카드마다 다른 포즈 → "내 카드"라는 소유감 상승 → 공유 욕구 폭발
- 뱃지 연동 포즈: "헤드헌터" → 조준 포즈, "클러치 머신" → 1v5 긴장감 있는 포즈
- 티어별 포즈 차등: 레디언트는 역동적인 포즈, 낮은 티어는 기본 포즈
- 랜덤 요소로 "다시 뽑기" 재미 부여 가능
- **주의**: Riot 팬 콘텐츠 정책 확인 필요, AI 생성 이미지 퀄리티 일관성 및 화풍 통일 유지 필수

### 3. 뱃지 시스템

- 스탯 기반으로 특별한 뱃지 부여
- 예시: "헤드헌터", "클러치 머신" 등
- 다양한 뱃지 종류로 수집 욕구 자극
- "나 클러치 머신 뱃지 떴어 ㅋㅋ" → 바이럴의 시작점

### 5. 시그니처 무기

- 가장 많이 킬한 무기를 카드에 아이콘으로 표시
- FUT 카드에서 국기 아이콘이 들어가는 것처럼 자연스러운 배치
- "반달 장인", "오퍼 원챔" 같은 무기 기반 태그도 가능
- 같은 티어, 같은 요원이라도 무기가 다르면 다른 카드 → 개성 강화

### 6. AI 피드백 한줄

- AI가 전적을 분석해 한줄 피드백 제공
- 예시: "1v1 상황에서 무적이지만, 초반 라운드에서 너무 공격적입니다"
- 재미 + 인사이트를 동시에 제공
- 스크린샷으로 공유하기 좋은 포맷

## 왜 바이럴 가능성이 있는가

1. **공유 본능 자극**: 카드 한 장으로 자기과시 → 친구에게 "너는?" → 자연스러운 바이럴 루프
2. **검증된 포맷**: FUT 카드 디자인은 전 세계적으로 인지도가 높은 포맷
3. **거대한 유저풀**: 발로란트 글로벌 유저 수 + 전적 사이트 유저가 그대로 타겟
4. **비교 욕구**: 랭크 게임 유저들은 본능적으로 비교하고 싶어함
5. **기존 전적 사이트와 공존**: 경쟁이 아닌 보완 관계

## 바이럴 극대화 전략

- 카드 이미지 다운로드/공유 버튼 필수
- OG 이미지로 링크 미리보기에 카드가 뜨도록 설정
- 친구와 카드 나란히 비교하는 기능
- 새 시즌마다 다시 만들러 오게끔 시즌/패치 연동

## 리스크

- Riot API 정책 변경에 대한 의존도
- 알고리즘 점수의 공정성/신뢰도에 대한 유저 피드백
- 초기 트래픽 확보를 위한 마케팅 필요 (레딧 r/VALORANT, 디스코드 커뮤니티 등)
