# 프로젝트 차별점

## 핵심 컨셉

- 전적 사이트가 아님. tracker.gg, op.gg 같은 분석 도구와 다른 카테고리.
- 전적 기반 FC FUT 카드 디자인을 생성하는 "자랑 도구".
- 디자인이 메인. 공유되는 건 이미지 한 장이고, 그 한 장이 예뻐야 퍼짐.

## 핵심 기능

### 1. FC 카드 디자인

- 발로란트 전적 데이터 기반 FC FUT 스타일 카드 생성
- 티어별 카드 디자인 차등. 높은 티어일수록 화려함.
- [[tier-scoring-algorithm]] 기반 알고리즘 점수 산출

### 2. AI 요원 이미지 생성 (NanoBanana2)

- 공식 일러스트 그대로 쓰면 모든 유저 카드가 동일. 개성 없음.
- NanoBanana2로 공식 화풍 유지하면서 다양한 포즈 생성
- 같은 요원이라도 카드마다 다른 포즈. 소유감 상승, 공유 욕구 자극.
- 뱃지 연동 포즈 가능. "헤드헌터" -> 조준 포즈, "클러치 머신" -> 1v5 포즈.
- 티어별 포즈 차등. 레디언트는 역동적, 낮은 티어는 기본 포즈.
- 랜덤 요소로 "다시 뽑기" 재미 부여 가능
- 주의: Riot 팬 콘텐츠 정책 확인 필요.
  AI 이미지 퀄리티 일관성 및 화풍 통일 유지 필요.

### 3. 뱃지 시스템

- 스탯 기반 뱃지 부여. 예: "헤드헌터", "클러치 머신"
- 다양한 종류로 수집 욕구 자극
- 뱃지 획득 자체가 공유 포인트가 됨

### 4. 시그니처 무기

- 가장 많이 킬한 무기를 카드에 아이콘으로 표시
- FUT 카드의 국기 아이콘처럼 자연스러운 배치
- "반달 장인", "오퍼 원챔" 같은 무기 기반 태그 가능
- 같은 티어, 같은 요원이라도 무기가 다르면 다른 카드. 개성 강화.

### 5. AI 피드백 한줄

- AI가 전적 분석 후 한줄 피드백 제공
- 예: "1v1 상황에서 무적이지만, 초반 라운드에서 너무 공격적임"
- 재미 + 인사이트 동시 제공
- 스크린샷 공유에 적합한 포맷

## 바이럴 가능성

- 카드 한 장으로 자기과시 -> "너는?" -> 자연스러운 바이럴 루프
- FUT 카드 디자인은 전 세계적으로 인지도 높은 포맷
- 발로란트 글로벌 유저 수 + 전적 사이트 유저가 그대로 타겟
- 랭크 게임 유저의 비교 욕구 자극
- 기존 전적 사이트와 경쟁이 아닌 보완 관계

## 바이럴 전략

- 카드 이미지 다운로드/공유 버튼 필수
- OG 이미지로 링크 미리보기에 카드 노출
- 친구 간 카드 비교 기능
- 시즌/패치 연동으로 재방문 유도

## 리스크

- Riot API 정책 변경 의존도
- 알고리즘 점수 공정성/신뢰도 이슈
- 초기 트래픽 확보 필요. 레딧 r/VALORANT, 디스코드 커뮤니티 등.
