# Code Review Style Guide

코딩 컨벤션과 규칙은 [CLAUDE.md](/CLAUDE.md) 따름.

### 리뷰 범위

- 이 styleguide.md에 명시된 규칙과 CLAUDE.md에 명시된 규칙만을 기준으로 리뷰할 것. 사전 학습된 지식이나 일반적인 베스트 프랙티스를 근거로 리뷰하지 말 것.
- 라이브러리 버전 리뷰 금지. dependabot으로 관리함.

### 리뷰 문체

- 한국어로 작성. "~함", "~추가", "~구현" 같은 간결한 문체 사용.
- 한 문장을 길게 쓰지 말고 끊어서 작성.
  - 좋은 예: "- 불필요한 리렌더 발생함, - useEffect 의존성 정리 필요"
  - 나쁜 예: "이 부분은 불필요한 리렌더가 발생하고 있어서 useEffect의 의존성을 정리해야 할 것 같습니다"
